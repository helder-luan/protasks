# Easy Tasks

Gerenciador de tarefas para organizar suas atividades diárias.

## Funcionalidades

- [x] Adicionar tarefas
- [x] Remover tarefas
- [x] Marcar tarefas como concluídas

## Tecnologias

- React
  - Hooks
  - Context API
- Styled Components
  - Tema Dark e Light
- Local Storage

## Como rodar o projeto

1. Clone o repositório

```bash
git clone https://github.com/helder-luan/protasks.git
```

2. Acesse a pasta do projeto

```bash
cd easytasks
```

3. Instale as dependências

```bash
npm install
```

4. Rode o projeto

```bash
npm run dev
```
